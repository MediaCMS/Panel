# MediaCMS Panel

### Панель керування

*Експериментальний проект* 

Сайт в розробці

##### Мета проекту:
1. Дослідження архітектури складних сайтів
2. Впровадження іноваційних технологій та вдосконалень
3. Технічна підтримка засобів масової інформації

##### Використання

1. Перейменувати файл налаштувань та заповнити його згідно опису \
`private/settings.template.php` > `private/settings.php` 
 
2. Надати дозвіл для запису на файл `private/log`

3. При потребі підключити необхідні модулі Apache та PHP
