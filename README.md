# MediaCMS Panel

### Панель керування

*Експериментальний проект* 

Сайт в розробці

##### Мета проекту
1. Дослідження архітектури складних сайтів
2. Впровадження іноваційних технологій та вдосконалень
3. Технічна підтримка засобів масової інформації

##### Використання

1. Створити БД, користувача (читання, запис та виклик процедур), імпортувати таблиці та процедури: \
`private/tables.sql` та `private/procedures.sql`

2. Створити копію файлу налаштувань з шаблону та заповнити його згідно опису: \
`private/settings.template.php` > `private/settings.php` 
 
3. Надати дозвіл для запису на файл `private/log`

При потребі підключити необхідні модулі Apache (rewrite) та PHP (intl)
