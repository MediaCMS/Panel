# MediaCMS Panel

### Панель керування MediaCMS
Експериментальний проект (*в розробці*)



##### Мета проекту:
1. Дослідження архітектури складних сайтів
2. Впровадження іноваційних технологій та вдосконалень
3. Технічна підтримка засобів масової інформації

##### Використання

1. Перейменувати файл налаштувань
`private/settings.template.php` > `private/settings.php` та заповнити його згідно опису

2. Надати дозвіл для запису на теку `private/logs`

3. При потребі підключити необхідні модулі Apache та PHP
 
##### Розповсюдження
Розповюдження дозволено тільки з метою ознайомлення для навчання
