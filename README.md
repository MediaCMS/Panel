# MediaCMS Panel

### Панель керування

*Експериментальний проект* 

Сайт в розробці

##### Мета проекту
1. Дослідження архітектури сайтів
2. Тестування іноваційних технологій та вдосконалень
3. Технічна підтримка засобів масової інформації

##### Середовище
1. Apache => 2.4 (+rewrite)
2. PHP => 7.2 (+intl)
3. MySQL => 5.7

##### Встановлення
1. Створити БД та користувача (читання, запис та виклик процедур), імпортувати таблиці, процедури та початкові дані:
\
`private/tables.sql` \
`private/procedures.sql` \
`private/data.sql`

2. Створити копію файлу налаштувань з шаблону та заповнити його згідно опису:
\
`private/settings.template.php` > `private/settings.php` 

3. Завантажити WYSIWYG-редактор TinyMCE => 5.0 з українською локалізацією в теку:
\
`public/tinymce`

##### Застереження
1. Проект в розробці та постійно оновлюється
2. Автор залишає за собою право в любий момент без попередження та пояснень як завгодно змінювати код
3. Не рекомендується використовувати для робочих проектів
