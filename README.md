# MediaCMS Panel

### Панель керування

*Експериментальний проект* 

Сайт в розробці

##### Мета проекту
1. Дослідження архітектури сайтів
2. Тестування іноваційних технологій та вдосконалень
3. Технічна підтримка засобів масової інформації


##### Встановлення
1. Встановити та налаштувати Apache => 2.4 (rewrite), PHP => 7.2 (intl) та MySQL => 5.7

2. Створити БД та користувача (читання, запис та виклик процедур), імпортувати таблиці, процедури та початкові дані:
\
`private/tables.sql` \
`private/procedures.sql` \
`private/data.sql`

3. Створити копію файлу налаштувань з шаблону та заповнити його згідно опису:
\
`private/settings.template.php` > `private/settings.php`
