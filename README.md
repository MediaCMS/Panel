# MediaCMS Panel

### Панель керування

*Експериментальний проект* 

Сайт в розробці

##### Мета проекту
1. Дослідження архітектури сайтів
2. Впровадження іноваційних технологій та вдосконалень
3. Технічна підтримка засобів масової інформації

##### Встановлення

1. Створити БД та користувача (читання, запис та виклик процедур), імпортувати таблиці та процедури: \
`private/tables.sql` та `private/procedures.sql`

2. Створити копію файлу налаштувань з шаблону та заповнити його згідно опису: \
`private/settings.template.php` > `private/settings.php` 
 
3. Надати дозвіл для запису на файл `private/log`

При потребі підключити необхідні модулі Apache2 (rewrite) та PHP7 (intl)

##### Застереження:
1. Проект в розробці та постійно оновлюється
2. Автор залишає за собою право в любий момент як завгодно змінювати код
3. Не рекомендується використовувати для робочих проектів
